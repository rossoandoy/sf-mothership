# SF Mothership プロダクトビジョン

## 1文定義

**Salesforce 画面を開いた瞬間に、「このレコード/この画面で今確認すべきこと」が3秒で分かる導入現場アシスタント。**

## ペルソナ

- Salesforce 導入コンサルタント / 開発者
- UAT 担当者
- 案件固有のオブジェクト・権限を毎日確認する現場メンバー

## 3つの体験（v0.3.0）

| 体験 | ユーザーの問い | タブ |
|------|--------------|------|
| 今すぐ | このレコード/画面で何が見えて、権限はどう？ | 今すぐ |
| 項目 | API名・型・編集可否は？ | 項目 |
| ガイド | UAT で何を確認すべき？ | ガイド |

## やらないこと

- SIR のような万能 API 操作ツール
- 常時オンな汎用 AI チャット
- 設定だらけの初回体験（デフォルトで動く）

## 母艦思想

全部入りの巨大ツールではなく、**画面文脈に応じた小さな答えを即座に出す基盤**。
案件ごとに Pack でガイド・宣言的ツールを差し替える。
