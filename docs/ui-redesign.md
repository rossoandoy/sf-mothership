# UI 再設計ワイヤーフレーム（v0.3.0）

## レイアウト

```
+--------------------------------------------------+
| SF Mothership  v0.3.0              [Sandbox]     |
+--------------------------------------------------+
| Account · 001xx... · レコードページ               |  <- ContextBar
+--------------------------------------------------+
| [今すぐ]  [項目]  [ガイド]                        |  <- ContextTabs
+--------------------------------------------------+
|                                                  |
|  (タブ内容 — 自動ロード)                          |
|                                                  |
|  今すぐ: レコード概要カード + アクセス診断カード    |
|  項目:  フィールドテーブル + 検索                  |
|  ガイド: UAT チェックリスト                       |
|                                                  |
+--------------------------------------------------+
| Pack: manabie          [もっと見る ▲]            |
+--------------------------------------------------+
```

## 「もっと見る」ドロワー

```
+--------------------------------------------------+
| 上級ツール                              [閉じる] |
+--------------------------------------------------+
| [操作] テストデータ作成                           |
| [AI]   ツール定義生成 (App Server 有効時のみ)     |
| [AI]   アクセス診断 AI説明                        |
| [AI]   レポート分析                               |
| [参照] 項目一覧 (宣言的)  ... Pack 由来           |
+--------------------------------------------------+
```

## 状態遷移

1. Side Panel 起動 → 「今すぐ」タブ自動ロード
2. タブ切替 → 該当タブのツールを自動実行
3. 「もっと見る」→ ドロワー表示 → ツール選択 → ToolPanel フルスクリーン
4. ToolPanel「戻る」→ タブ UI に復帰

## 設計原則

1. デフォルトで答えが出る（ツール選択不要）
2. Write / AI は二次導線
3. Loading / Error / Empty をタブごとに統一
